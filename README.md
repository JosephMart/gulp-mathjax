# gulp-mathjax

Custom gulp plugin for use with mathjax

## Installation

```sh
npm install gulp-mathjax --save
yarn add gulp-mathjax
```

## Usage

### Javascript

```javascript
var gulp_mathjax = require("gulp-mathjax");
```

### TypeScript

```typescript
import { gulp_mathjax } from "gulp-mathjax";
```

## Test

```sh
npm run test
```
